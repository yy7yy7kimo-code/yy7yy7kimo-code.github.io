公開支援管道

若你在使用「秘密影音」時遇到問題，請透過以下正式支援入口回報：

  • GitHub Issues 支援入口
  • 隱私權政策

回報時建議提供裝置型號、iOS 版本、App 版本、問題步驟與畫面狀況，以便追蹤與回覆。

常見問題

無法解鎖

  • 確認你已完成初始密碼設定。
  • 確認輸入的密碼正確。
  • 若仍無法解鎖，請依 App 內重設流程重新設定；重設可能會清除本機設定或狀態。

內容沒有出現

  • 檢查網路連線是否正常。
  • 確認目標網站或資料來源可正常開啟。
  • 重新整理或重新啟動 App 再試一次。

廣告或廣告同意畫面異常

  • 檢查網路連線是否正常。
  • 若所在地區需要同意，請依 App 內顯示的同意畫面或廣告隱私選項調整。
  • 廣告未出現通常不影響核心播放與瀏覽功能。

私密模式會做什麼

私密模式用於降低搜尋與歷史狀態保存。私密模式不等同匿名上網，也不會改變第三方網站或網路服務的資料處理方式。

營運維護確認

本支援頁、隱私權政策、App Store metadata、App Privacy labels 與實際 App 行為需同步維護。
