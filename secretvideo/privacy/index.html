秘密影音隱私權政策

最後更新日期：2026-05-04

本隱私權政策說明「秘密影音」（以下稱「本 App」）如何處理資料。

我們處理哪些資料

依你啟用的功能，本 App 可能會處理以下類型的資料：

  • 裝置與應用程式資訊：版本號、裝置狀態、錯誤日誌、診斷資訊。
  • 本機設定：密碼狀態、私密模式開關、語言、偏好設定。
  • 使用者內容：你輸入、匯入、瀏覽或儲存的內容。
  • 網路與廣告相關資訊：顯示廣告所需的有限資訊，以及與廣告 SDK 運作相關的診斷資料。

我們如何使用這些資料

  • 讓 App 正常運作。
  • 保存你的本機設定與狀態。
  • 提供你主動使用的功能。
  • 偵測錯誤、改善穩定性與疑難排解。
  • 顯示廣告與維持 App 的營運。

廣告、同意與追蹤

本 App 可能顯示由 Google Mobile Ads SDK 提供的廣告內容。為了載入、限制、衡量與改善廣告，廣告 SDK 可能依 Google 服務條款、裝置設定與同意狀態處理裝置識別碼、廣告資料、產品互動與診斷資訊。

本 App 使用 Google User Messaging Platform（UMP）請求與呈現必要的廣告同意流程；廣告請求只會在同意狀態允許請求廣告後啟動。若你所在地區或裝置狀態需要同意，請依 App 內顯示的同意畫面、App 內的廣告隱私選項或系統設定調整。

本 App 不主動請求 Apple 追蹤授權，也不以裝置廣告識別碼作為本 App 的必要廣告啟動條件。廣告請求會依 Google Mobile Ads / UMP、你的所在地區同意狀態與系統設定，以可用的非追蹤或同意後方式載入。

資料是否分享給第三方

只有在以下情況才可能傳送資料給第三方：

  • 廣告 SDK 載入、限制、衡量與顯示廣告所需的服務。
  • 你主動匯入、同步或使用外部資料來源。
  • 這些服務本身就需要資料傳輸才能完成功能。

我們不會在未經你主動啟用或授權的情況下，將資料用於與功能無關的用途。

資料保存與刪除

  • 大部分設定與狀態會保存在裝置本機或 Keychain。
  • 若你清除 App、重設設定或停用功能，對應資料會隨之移除或失效。
  • 若你刪除 App，裝置上的本機資料通常會一併移除。

安全性

我們會盡合理努力保護資料安全，但任何網路傳輸都無法保證百分之百安全。

兒童隱私

本 App 並非專門面向兒童設計。若你認為有未經允許的兒童資料蒐集，請透過支援頁聯絡我們以便處理。

你的選擇

  • 停用私密模式或相關功能。
  • 使用 App 內的廣告隱私選項或系統設定調整同意狀態。
  • 清除 App 設定。
  • 刪除 App。

聯絡方式

請透過秘密影音支援頁回報問題或提出詢問。
